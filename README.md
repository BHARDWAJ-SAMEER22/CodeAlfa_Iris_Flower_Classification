# Iris Flower Classification

This project is part of the CodeAlpha Data Science Internship.

## Objective
To classify Iris flowers into Setosa, Versicolor, and Virginica using machine learning.

## Dataset
Built-in Iris dataset from scikit-learn.

## Model Used
Logistic Regression

## Result
Model achieved 96% accuracy.
